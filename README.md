# PokePrint Admin

Order tracking, inventory, and design management.

## Files
- `track-order-v2.html` - Order tracker (main page)
- `inventory.html` - Filament inventory
- `designs.html` - Design management
- `track-order.html` - Legacy tracker (backup)
- `app.js` - Legacy app (backup)
- `n3d-sync.js` - N3D sync utilities

## Deploy
```bash
firebase hosting:sites:create admin
firebase deploy --only hosting:admin
```
